# Design do Sistema — WA Fin Ctrl

> Base: [./README.md](./README.md)

Esta fase detalha como os requisitos e a arquitetura se transformam em componentes concretos. Os artefatos descrevem estruturas Python/TypeScript, regras de interação, layouts e fluxos operacionais.

Documentos desta seção:
- [Design Geral](design-geral-spec.md) — princípios de design, camadas, dependências entre módulos e diretrizes de estilo.
- [Componentes](componentes-spec.md) — inventário de componentes (CLI, serviços Python, módulos FastAPI, páginas React) com responsabilidades claras.
- [Fluxos](fluxos-spec.md) — sequências ponta a ponta (processamento incremental, correção, sincronização cloud, auditoria).

Os diagramas podem ser representados em Mermaid, ASCII ou descrições textuais para manter versionamento simples. Qualquer alteração significativa deve atualizar esta fase antes de iniciar a implementação.
